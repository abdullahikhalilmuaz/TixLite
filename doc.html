User Authentication

POST /api/signup
Register a new user account

Request Body:

firstname string User's first name
lastname string User's last name
username string Unique username
email string User's email address
password string User's password

Event Management

POST /api/events
Create a new event with image upload

Request Body (multipart/form-data):

title string Event title
description string Event description
date string Event date
time string Event time
location string Event location
tickets number Available tickets
image file Event image (max 5MB)
accountName string Payment account name
accountNumber string Payment account number
bankName string Bank name
GET /api/events
Get all events
GET /api/events/:id
Get event by ID

URL Parameters:

id string Event ID
GET /api/events/admin/:adminId
Get events created by specific admin

URL Parameters:

adminId string Admin ID

Ticket Management

POST /api/events/:eventId/tickets
Create a ticket request with payment proof

Request Body (multipart/form-data):

userId string User ID
userDetails string JSON string of user details
paymentProof file Payment proof image (max 5MB)
POST /api/events/tickets/:ticketId/approve
Approve a ticket request
POST /api/events/tickets/:ticketId/reject
Reject a ticket request

Request Body:

rejectionReason string Reason for rejection
GET /api/events/tickets/:ticketId
Get ticket information
GET /api/events/tickets/user/:userId
Get all tickets for a specific user
GET /api/events/tickets/admin/:adminId
Get all tickets for events created by a specific admin
